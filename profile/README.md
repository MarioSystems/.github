## About

MarioSystems is a personal software engineering organization focused on:

* Backend development
* Security-oriented systems
* Authentication and authorization
* Scalable architectures
* AI-assisted security analysis
* DevOps and infrastructure concepts

The goal of these projects is to simulate real-world enterprise environments while improving software engineering, architecture and infrastructure skills.

* Secure Route Guards
* Request Validation

* Modular Architecture
* Dockerized Environment

* NestJS
* TypeScript
